# Spotify Data Engineering – Databricks

Proyecto de Data Engineering end-to-end desarrollado en Databricks utilizando PySpark y arquitectura Lakehouse.

## Arquitectura
Bronze → Silver → Gold

## Flujo del proyecto
- Ingesta de datos desde Volumes (Unity Catalog)
- Persistencia en tablas Bronze
- Limpieza y transformaciones en Silver
- Modelado analítico en Gold
- Visualización mediante Dashboard

## Tecnologías
PySpark, Databricks, Delta Lake, Unity Catalog
